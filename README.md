# DXD NextGen Website

This is a one-page static website for DXD NextGen — Robotics, AI and Coding education/lab implementation.

## Files
- `index.html` — main website
- `styles.css` — design and responsive layout
- `script.js` — mobile menu and WhatsApp enquiry generator

## Before publishing
1. Open `script.js`
2. Replace:
   `const WHATSAPP_NUMBER = "91XXXXXXXXXX";`
   with your real WhatsApp number in international format.

Example:
`const WHATSAPP_NUMBER = "919876543210";`

3. In `index.html`, update any wording, contact details, package details and legal/reference wording based on your real operations.
4. Do not add fake school names, fake results, fake awards or fake certifications. Add real proof as you create it.

## GitHub Pages
Upload these files directly to your repository root:
- index.html
- styles.css
- script.js

Then enable GitHub Pages from repository settings.
